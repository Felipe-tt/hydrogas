Política de Privacidade

Última atualização: abril de 2025

O aplicativo HidroGás é um sistema de gestão de leituras de água e gás destinado a condomínios. Esta política descreve como os dados são coletados, usados e protegidos.

1. Dados coletados

  • Administrador: nome de usuário e senha (armazenados como hash Argon2id, nunca em texto puro).
  • Moradores: nenhum dado pessoal é coletado. O acesso é feito por link/token anônimo.
  • Leituras: valores de consumo de água e gás por unidade, sem vinculação a CPF ou dados pessoais.
  • Biometria: dados de digital/Face ID são processados inteiramente no dispositivo via WebAuthn. Nenhum dado biométrico é enviado ou armazenado em servidores.

2. Uso dos dados

Os dados são usados exclusivamente para:

  • Calcular o consumo e o custo de água e gás por apartamento;
  • Gerar relatórios mensais para o síndico;
  • Permitir que moradores consultem seu próprio histórico de consumo.

3. Compartilhamento

Os dados não são compartilhados, vendidos ou repassados a terceiros. O único serviço externo utilizado é o Firebase (Google), para autenticação e armazenamento de dados, sujeito à Política de Privacidade do Google.

4. Segurança

O aplicativo adota as seguintes medidas de segurança:

  • Senhas com hash Argon2id (64 MiB, 3 iterações);
  • Autenticação biométrica via WebAuthn processada localmente no dispositivo;
  • Rate limiting em todas as operações sensíveis;
  • Regras de acesso no Firebase que impedem leitura de dados administrativos sem autenticação válida;
  • Comunicação exclusivamente via HTTPS.

5. Retenção de dados

Os dados de leitura são mantidos enquanto o condomínio utilizar o sistema. O administrador pode excluir dados a qualquer momento pelo painel.
